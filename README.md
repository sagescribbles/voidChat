# VoidChat

VoidChat is an anonymous real-time communication platform that supports text and voice chat using WebRTC.

## Features
- Anonymous chat rooms
- Real-time messaging
- WebRTC voice communication
- TURN/STUN connectivity for global connections

## Tech Stack
- React
- TypeScript
- Vite
- Supabase (signaling + backend)
- WebRTC
- Tailwind CSS

## Installation

```bash
git clone https://github.com/Kuladeep-M-N/voidChat.git
cd voidChat
npm install
npm run dev
